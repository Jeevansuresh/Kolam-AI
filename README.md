Kolam AI is an AI-powered virtual front desk prototype built for serviced apartments. The goal is to automate repetitive front desk tasks and improve guest experiences using smart, data-driven automation. This prototype demonstrates the potential of AI in the hospitality industry, offering seamless guest interaction and backend operational insights.

Project Structure:

app.py: The main Python application that powers the backend of the Kolam AI assistant, handling requests and processing logic.

templates/: Contains HTML templates that define the structure of the front-end interface for user interactions.

index.html: The landing page that offers initial user interaction with the AI assistant.

Features:
Basic framework for automating front desk tasks

Template for integrating with web-based interfaces

Scalable architecture for adding more features

Future Enhancements:
Integration with real-time booking systems

NLP-based guest feedback handling

Room service and concierge functionalities
